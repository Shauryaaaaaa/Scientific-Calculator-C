# Scientific Calculator (C-Language)
A high-performance command-line scientific calculator built using C, focusing on modular logic and precise mathematical computations.

## 🚀 Key Features
* **Advanced Arithmetic:** Support for basic operations and complex powers using `math.h`.
* **Trigonometric Functions:** Real-time calculation of Sin, Cos, and Tan values.
* **Logarithmic Logic:** Implementation of natural and base-10 logarithms.
* **Input Parsing:** Uses `string.h` for robust command handling and user interface interaction.
* **Error Handling:** Built-in checks for mathematical errors like division by zero or invalid square roots.

## 🛠️ Tech Stack
* **Language:** C (C11 Standard)
* **Libraries:** `stdio.h`, `math.h`, `string.h`
* **Environment:** Developed and tested in VS Code
